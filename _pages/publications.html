Research and Project Portfolio

This portfolio presents selected academic, research, and development work by Abhijit Kumar Ghosh, Lecturer in the Department of Computer Science and Engineering at the European University of Bangladesh.

The portfolio reflects work in Machine Learning, Deep Learning, Computer Vision, Medical Image Analysis, Trustworthy AI, Multimodal AI, Time-Series Forecasting, Applied Data Science, and AI-based decision support systems.

Selected Projects

AI-Based Grade Prediction System

Type: Machine Learning / Educational Data Mining

This project focused on predicting student academic performance using educational indicators such as quizzes, attendance, assignments, and class participation. The purpose of the system was to identify students who may need early academic support.

  • Built a machine learning-based grade prediction system.
  • Used academic indicators to analyze student performance patterns.
  • Applied Python, scikit-learn, and SQL Server for data analysis and model development.
  • Designed the project to support data-driven academic decision-making.

Machine Learning Educational Data Mining Python Scikit-learn SQL Server

Bangla Language Developing System

Type: Language Learning / Educational Technology

This project proposed a dedicated Bangla language learning and assessment platform. The system was designed to support students, learners, and job-sector examination candidates in Bangladesh.

  • Designed the idea for a Bangla-focused learning and assessment platform.
  • Focused on education and job-sector examination preparation.
  • Proposed structured support for language learning and evaluation.

Bangla Language Educational Technology Assessment System Learning Platform

Fire Alarm System in Farm

Type: Embedded System / Safety Monitoring

This project involved developing an Arduino Nano-based fire alarm system for farm environments. The goal was to support early fire hazard detection and improve safety response in agricultural settings.

  • Developed an early warning fire alarm system using Arduino Nano.
  • Focused on farm and agricultural safety monitoring.
  • Designed the system for simple and practical fire hazard detection.

Arduino Nano Embedded System IoT Safety Monitoring

Research-Oriented Works

Histo-MExNet: Trustworthy Breast Cancer Histopathology Analysis

Research Area: Medical Image Analysis / Trustworthy AI

This work focuses on real-world, cross-magnification, and trustworthy breast cancer histopathology analysis. It reflects my research interest in medical imaging, deep learning, and reliable AI systems for healthcare applications.

Medical Image Analysis Breast Cancer Deep Learning Trustworthy AI

Swin Transformer-Based Gallbladder Disease Classification

Research Area: Computer Vision / Medical Imaging

This research work explored multi-class gallbladder disease classification using a Swin Transformer-based approach and compared it with lightweight convolutional neural network models.

Swin Transformer Computer Vision Disease Classification Medical AI

Multimodal Phishing Detection Using VLM and LLM Reasoning

Research Area: Multimodal AI / Cybersecurity

This work investigated phishing detection and deletion using small vision-language models and large language model reasoning. It connects multimodal AI with cybersecurity and trustworthy decision-making.
